JAP - Just another incomplete P2P
=======
### Dependencies
* nodejs
* npm

### Install
```bash
git clone https://github.com/astroza/jap
cd jap
npm install
export JAP=$(pwd)
```
### Usage
```bash
mkdir $HOME/jap.data
cd $HOME/jap.data
node $JAP/server.js
```

Now, you can share files in **$HOME/jap.data/storage/share** directory.
